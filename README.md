# Alessanna / MAVE

Статический сайт с двумя проектами:

- `alessanna.html` — Alessanna на русском
- `alessanna-et.html` — Alessanna на эстонском
- `mawe.html` — MAVE на русском
- `mawe-et.html` — MAVE на эстонском
- `index.html` — стартовая страница

Основные файлы:

- `styles.css`
- `script.js`
- `mawe-logo.svg`
- `assets/`

Для публикации на GitHub Pages загрузите в репозиторий все файлы из этой папки.
